How Do I Obtain the Current Time Using an Expression?
=====================================================

Symptom
-------

How do I obtain the current time using an expression?

Solution
--------

You can use the **DateUtils.format(${timestamp()},"yyyy-MM-dd HH:mm:ss")** expression on the **Map Field** page to obtain the current time.
